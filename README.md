## Overview
This project is a simple AI-powered feature built for an internship platform called **InternHub**.  
It analyzes a student profile and an internship job description to provide intelligent insights such as internship match summary, skill gap analysis, recommendations, and an ATS confidence score.

The project focuses on **AI reasoning and prompt-based logic**, not UI or complex backend systems.

## Features
- Internship match summary
- Skill gap explanation
- Personalized recommendations
- ATS-style confidence score
- Simple, explainable AI logic
